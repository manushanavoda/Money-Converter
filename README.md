# Money-Converter

money Converter is a modern, professional-grade desktop currency converter application built with **Python**. It provides real-time exchange rate data with a sleek, user-friendly interface that supports both Light and Dark modes.

## âœ¨ Features

* **Real-Time Data:** Fetches live currency rates using an external API.
* **Dual Themes:** Seamlessly switch between **Light Mode** and **Dark Mode**.
* **Clean UI/UX:** Built with a professional white and green color palette for a modern feel.
* **Smart Swap:** Easily swap "From" and "To" currencies with a single click.
* **Copy to Clipboard:** Quickly copy results to use in other applications.
* **Live Rate Info:** Displays the conversion rate for 1 unit of the base currency.
* **Offline Handling:** Gracefully handles internet connection errors and invalid inputs.
